== Description ==

Google XML Sitemaps Automatically Generate and Update Sitemap in XML format.
Notify google,bing and yahoo on updates.
According to the updated google sitemap specification.

== Installation ==

1. Upload the `hide-title` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
